mobilecheck
===========

jquery plugin for check mobile or pc


How to used

1. install plugin
  <script src='jquery.mobilecheck.min.js'></script>
  
2. fn used

  mobilecheck();
 
example
  <script>
  
  var hello = mobilecheck() ? 'hello this is pc' : 'hello this is mobile';
  
  alert(hello);
  </script>
