mobilecheck
===========

jquery plugin for check mobile or pc


How to used

1. install plugin

  &lt;script src=&#39;jquery.mobilecheck.min.js&#39;&gt;&lt;/script&gt;
  
2. fn used

  mobilecheck();
 
example:

  &lt;script&gt;
  
  var hello = mobilecheck() ? 'hello this is pc' : 'hello this is mobile';
  
  alert(hello);
  &lt;/script&gt;
